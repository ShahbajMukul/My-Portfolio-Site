Course Management Software

This is an advanced Windows Forms application designed to simplify the process of managing college courses. Developed in C#, this system interfaces seamlessly with SQL Server to offer robust and efficient data management capabilities. It boasts a user-friendly interface that allows for the easy addition, modification, and deletion of courses, instructors, sections, students, and grades. This project was a deep dive into the world of database management, enhancing my proficiency in SQL and back-end development.

Instagram Astronomy Bot

Stargazing enters the social media universe with my Instagram Astronomy Bot. This Python-powered automaton brings the cosmos closer to astronomy enthusiasts by daily sharing the wonders of the universe. It harnesses the power of NASA's API to fetch the Astronomy Picture of the Day, combining it with captivating captions that spark curiosity and wonder. Hosted on Railway, the bot ensures uninterrupted celestial updates, inviting followers to embark on a virtual odyssey through the stars with just a scroll. Engage with the bot and let your Instagram feed become a gateway to the cosmic ballet.
